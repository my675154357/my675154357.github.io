关于此博客搭建

  • 前提

    最好有linux和git的操作经验，掌握常用的命令

  • 搭建

    参照简书上搭建个人博客-hexo+github详细完整步骤来搭建的。很快，十几分钟就搞定了，亲测可用。过程中基本没遇到什么大问题，有个地方要注意。第五步的第4小步的命令：

    npm install hexo-deployer-git  --save

    命令贴的有问题，install 后面的空格空错地方了。整个搞定之后就可以愉快的玩耍了。

搭建博客无需购买 云服务器、域名、DNS解析，完全免费，只需要注册github账号就好了。博客内容需要使用markdown语法编辑，刚好可以练习markdown。
最后说一下，每次遇到问题百度，出来很多参考结果，但有不少是雷同和无效甚至无意义的答案。需要自己取一个个尝试，比较浪费时间。这里比较推荐 简书 ，我觉得在简书上给出的答案都比较优质。个人感觉百度的时候不妨在问题后面加上 简书 去搜索答案，会相对高效一些。

新建或者修改了博客内容后

在博客目录下使用git bash，我这里博客目录是hexo

hexo new “文章标题名称” //新建文章

hexo clean //清理（非必须），在遇到问题时可尝试先clean再generate

hexo generate //将文章生成为博客的一部分
hexo g //同上，generate的简写

hexo server //本地预览（非必须）

hexo deploy //推送到GitHub
hexo d //同上，deploy的简写
